# automata-theme

An Emacs color theme inspired by the visual identity of
[NieR:Automata](https://www.platinumgames.com/games/nier-automata),
the 2017 action RPG directed by YOKO TARO and developed by PlatinumGames.

## Installation

Not yet available on MELPA. Manual installation only.

### Manual

Clone the repository:

```
git clone https://github.com/tichelmorres/automata-theme.git
```

Add the path to your Emacs config:

```elisp
(add-to-list 'custom-theme-load-path
             "/path/to/automata-theme/")
```

Then use `M-x customize-themes` to activate it.

## Contribution

The theme covers a large, yet limited, amount of modes.
If something looks wrong, add the appropriate faces and send a pull request.
